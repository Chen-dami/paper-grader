## 一、快速开始（只需 3 步）

### 方式 A：双击安装（推荐 · 新手友好）

```
1. 解压 ai-grader.zip 到任意目录
2. 双击 一键安装.bat → 等待 3-5 分钟
3. 配置 API Key（见下文）
4. 双击 启动阅卷系统.bat → 浏览器自动打开
```

### 方式 B：命令行安装（适合开发者）

```bash
# 1. 克隆仓库
git clone <仓库地址> ai-grader
cd ai-grader

# 2. 创建环境 + 安装依赖
python -m venv .venv
.venv\Scripts\python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 3. 启动
.venv\Scripts\streamlit run app.py
```

## 二、API Key 配置

### 获取 Key

| 平台 | 地址 | 模型 | 用途 | 费用 |
|------|------|------|------|------|
| **智谱** | [open.bigmodel.cn](https://open.bigmodel.cn) | glm-4v-flash | 视觉评分（看图） | **免费** |
| DeepSeek | [platform.deepseek.com](https://platform.deepseek.com) | deepseek-chat | 文本评分 | ¥1/百万 tokens |
| 阿里通义 | [dashscope.console.aliyun.com](https://dashscope.console.aliyun.com) | qwen-vl-max | 视觉备选 | ¥3/百万 tokens |
| OpenAI | [platform.openai.com](https://platform.openai.com) | gpt-4o-mini | 视觉+文本 | $0.15/百万 tokens |

### 配置方式（二选一）

**方式 A：环境变量（推荐）**
```batch
# 打开 cmd，执行：
setx ZHIPU_KEY "你的智谱API_Key"
setx DEEPSEEK_KEY "你的DeepSeek_API_Key"

# 配置后需重新打开终端窗口生效
```

**方式 B：.env 文件**
```
1. 将 .env.example 改名为 .env
2. 编辑 .env，填入你的 Key
```

## 三、各脚本说明

| 脚本 | 功能 | 何时用 |
|------|------|--------|
| `一键安装.bat` | 检测环境 + 安装 Python/Git + 创建 venv + 安装依赖 + 引导配置 Key | **第一次用** |

## 四、多机器部署

### U 盘 / 网盘分发

```
1. 复制整个 ai-grader/ 文件夹到目标电脑
2. 双击 一键安装.bat 安装环境
3. 配好 API Key
4. 双击 启动阅卷系统.bat
```

> 注意：`.venv/` 不需要复制，每台机器用 `一键安装.bat` 重建。`data/` 和 `output/` 包含试卷和成绩，需要保留。

### Git 更新流程

```bash
# 作者端：推送更新
git add . && git commit -m "新功能" && git push

# 用户端：拉取更新
双击 更新.bat
```

## 六、常见问题

**Q: 双击 .bat 闪退？**
A: 在脚本所在目录的地址栏输入 `cmd` 回车，手动运行脚本名，就能看到错误信息。

**Q: pip 安装慢 / 超时？**
A: 脚本已配置清华镜像。如仍失败，手动执行：
```batch
.venv\Scripts\pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
```

**Q: 提示 "模型不可用"？**
A: 检查 API Key 是否配置：`echo %ZHIPU_KEY%`。在 Web 界面「配置 → 模型路由」可查看各模型状态。

**Q: 智谱视觉不工作，返回 fallback？**
A: 确认 `ZHIPU_KEY` 已设置。`glm-4v-flash` 免费版可能有频率限制，批量阅卷时稍等片刻即可。

**Q: 端口 8501 被占用？**
A: `启动阅卷系统.bat` 会自动释放端口。如果仍占用，修改 `config.yaml` 或在启动命令中指定：`streamlit run app.py --server.port 8502`
